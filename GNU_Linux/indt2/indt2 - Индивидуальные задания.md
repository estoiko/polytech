## Индивидуальные практические задания по использованию команды grep и конвейеров обработки

### Задание 2

Найти корректные MAC-адреса (HH:HH:HH:HH:HH:HH, шестнадцатерично, регистр любой). Подсчитать количество адресов по первым трём байтам (OUI) и вывести топ-5. [^1]

Примеры:

```
00:1A:2B:3C:4D:5E — Да
GG:11:22:33:44:55 — Нет
```

```bash
grep -Eio "([0-9A-F]{2}:){5}([0-9A-F]{2})" mac.txt | cut -d: -f 1-3 | uniq -c | sort -nr | head -n 5
```

[^1]: [regex101](https://regex101.com/r/7rKJJ1/1)
